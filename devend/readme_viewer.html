System Overview

DeveloperNotes: Contains standard operating procedures and technical documentation for frontend orchestration.
DevEnd Portal: A restricted management layer for system monitoring, log tracking, and live configuration updates.
Frontend Logic: Modular JavaScript handlers that sync with Google Apps Script (GAS) for serverless form processing.
